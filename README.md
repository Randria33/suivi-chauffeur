# SuiviChauffeur - Micro-SaaS pour chauffeurs independants

## Concept
Application web simple permettant aux chauffeurs-livreurs independants de suivre:
- Leurs revenus journaliers/hebdomadaires/mensuels
- Leurs kilometres
- Leurs depenses (carburant, peages, repas)
- Leurs documents (contrats, factures)

## Budget de demarrage (500€)
- Hebergement VPS (1 an): ~180€
- Nom de domaine (1 an): ~10€
- Base de donnees Supabase (gratuit): 0€
- Templates UI (Tailwind): 0€
- Marketing initial (Meta Ads): ~200€
- Frais bancaires Stripe: 0€
- **Total: ~390€** (reste 110€ de marge)

## Modele economique
- Abonnement: 9,99€/mois par chauffeur
- Objectif M1: 5 clients = 50€/mois
- Objectif M6: 30 clients = 300€/mois
- Seuil rentabilite: 18 clients

## Pourquoi ca marche
- 350+ chauffeurs independants en IDF (DPD, GLS, DHL)
- Aucun outil simple dedie a cette cible
- Zou connait le marche (RZ Transport)
- Deploiement rapide (MVP en 2 semaines)
