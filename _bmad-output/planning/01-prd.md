# PRD - SuiviChauffeur v1.0

## Functional Requirements
FR1: Inscription/connexion email
FR2: Saisie quotidienne (km debut/fin, carburant, peages, repas, colis, heures)
FR3: Auto-calcul (km parcourus, total depenses, gains estimes, marge)
FR4: Tableau de bord (gains mois, stats jour, dernieres saisies)
FR5: Historique (semaine/mois/annee) avec totaux
FR6: Export rapport texte
FR7: Mode hors-ligne (PWA)

## Non-Functional
NFR1: Saisie < 30 secondes
NFR2: Installation < 2Mo
NFR3: Fonctionne sans internet
NFR4: Donnees persistantes (localStorage)
