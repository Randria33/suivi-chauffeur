# UX Design - SuiviChauffeur

## Parcours utilisateur
1. Arrivee → Page d'accueil avec CTA "Commencer"
2. Inscription → Email + mot de passe + nom
3. Dashboard → Vue d'ensemble du mois en cours
4. Saisie → Formulaire rapide (< 30s)
5. Historique → Consultation et export

## Ecrans
1. Login/Register (splash screen)
2. Dashboard (gains mois, 4 stats rapides, dernieres saisies)
3. Saisie (formulaire avec recap auto)
4. Historique (filtres + tableau)
5. Export (bouton generation rapport)

## Design principles
- Mobile-first (PWA)
- Gros boutons (doigts)
- Contrastes eleves (lisibilite au soleil)
- Pas de scroll horizontal
